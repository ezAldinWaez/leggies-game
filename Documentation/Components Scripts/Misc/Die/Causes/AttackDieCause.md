# AttackDieCause

## Why it was written

To make Attack kill.
___

## What it does

It Implements [[DieCause]], isDieCauseLethal compares attack times, OnTriggerEnter2D calls OnDetectedDieCause if it hits anything that has the [[Attack]] Component.
___

## How to use it

Attach to what would die by Attack, also attach a [[DieMethod]] and link it with this.
