# FlyDieMethod

## Why it was written

To let stuff fly booom on death.
___

## What it does

It implements [[DieMethod]], making Die add a random force, in a random direction, with a random power--flyPower, a SerializeField. It then invokes DeleteObject after one second.
___

## How to use it

Attach to what you want to fly away on death, link with a [[DieCause]].
