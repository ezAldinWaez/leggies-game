# ThrowDieCause

## Why it was written

To let throws kill.
___

## What it does

It implements [[DieCause]], isDieCauseLethal always returns true, OnDetectedDieCause is called OnCollisionEnter2D with an object that has the [[Throwable]] Component.
___

## How to use it

Attach to what would die by throws, alongside a [[DieMethod]] and link with this.
