# InputName

## Why it was written

To store the names of the Input keys. Like, the UP key, the ATTACK key, and all of that. Just to have them in one place, and to have auto-complete on them.
___

## What it does

It holds an enum called InputName.
___

## How to use it

Whenever you need a new Input key, define its name here. So, whenever you need to bind that key to something, use its name.
