# DieAccessory

## Why it was written

To allow for stuff to happen on death that do not relate to the mechanism of death, but are extras to it.
___

## What it does

An abstract class. It subscribes an abstract void AccessoryAction to [[Die]]'s event, OnDying.
___

## How to use it

Write a class that implements it to add an accessory to death.
