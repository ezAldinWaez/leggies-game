# LeggiesLibrary

## Why it was written

Because we want to keep a collection of useful things here: functions and classes used all over the place. Instead of repeating them, we can have them here.
___

## What it does

It has the following classes:
- LeggiesMath: It has two methods:
	- ShakeVector3
	- RandomDirectionUnitVector2.
- LeggiesSounds: It has a method to play sound from audio clip; it makes a new game object to play the audio and destroy it in the end.
- LeggiesCameraOffset: It has the offset work needed for [[CameraFollow]].
___

## How to use it

See what you like from what it offers and use the namespace.
