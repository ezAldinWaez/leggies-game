# DieMethod

## Why it was written

Because there might be new ways to die. For new ways, you can just create a class that inherits from this. Like, written to make code cleaner and to collect the code in common between all methods of death.
___

## What it does

It provides an abstract Die method in which you'd implement the method of death. Also DeleteObject, which just destroys the object.
___

## How to use it

Just write classes that inherits from this.
